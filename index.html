What makes Node.js different ?

It allows ...

  • Event-Driven programing
  • Non-blocking I/O
  • Asynchronous programming


... and also, it's supposed to be fast because it's based on Google's V8


Google's V8 ?


  • It's a Javascript engine
  • It's written in C/C++ (not that anyone cares ...)
  • It implements ECMAScript (5th edition of ECMA 262)
  • It runs on Windows / Mac OS X / Linux (IA-32, X64, ARM)

Example of use


var http = require('http');
http.createServer(function (req, res) {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Hello World\n');
}).listen(1337, '127.0.0.1');
console.log('Server running at http://127.0.0.1:1337/');
